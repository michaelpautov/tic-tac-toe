# Tic Tac Toe
You can see demo https://tic-tac-toe-michael-kg.herokuapp.com/

## Description
Unbeatable tic-tac-toe game on a 3x3 board.

## Requirements
node 6.11.0+ <br>
npm 3.10.10+ <br>
typescript 2.0.6+ <br>

## Installation
npm install 

## Start
npm start

You can see your application by 'http://localhost:4200/' url

## Tests
npm run test
